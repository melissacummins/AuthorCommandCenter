Produce one paste-ready ad variation fast — for authors who already have a Book Analysis
(Story Overview, avatars, hooks) and just need a quick ad.

## When to Use This

- The author needs a quick ad for a flash sale, new creative, or time-sensitive promo
- The full 5-variation package is overkill for what they need right now
- They've already run `/setup-books` and have everything set up

## Workflow

1. **Check for existing Book Analysis.** Look for a saved Book Analysis file in the author's
   working directory, or check the current conversation. If none exists, tell the author to
   run `/setup-books` first.

2. **Ask three things:**
   - Which hook do you want to use? (Show the hook menu from the Book Analysis, or ask if
     they have a new one)
   - Which avatar are you targeting? (Show the avatar list, or ask who they're trying to reach)
   - What's the creative? (Image, video, or copy-first?)

3. **Write one complete paste-ready ad block:**
   - Hook + expansion targeting the chosen avatar
   - Trope list
   - Social proof
   - CTA
   - All in a single block the author copies straight into Ads Manager

4. **Write 2 headlines** that pair well with this specific variation.

5. **If the author doesn't already have an SEO block**, write one. If they do, remind them
   to use the existing one.

6. **Done.** No 5-variation package, no full headline pool, no lengthy process. One ad, ready
   to paste.
