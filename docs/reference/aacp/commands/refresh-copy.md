Generate fresh ad copy variations for a book that already has existing copy. Read the existing
ad copy document, keep the static blocks (trope list, social proof, CTA, SEO block), and
write NEW hook + expansion variations.

## Workflow

1. **Find the existing ad copy** for the book. Check the author's working directory for
   previous ad copy documents.

2. **Read the existing copy** and identify:
   - Which hooks and frameworks have already been used
   - The static blocks (trope list, social proof, CTA)
   - The SEO description block
   - The headline pool

3. **Read the book's avatar profiles** (if they exist) and the book profile.

4. **Write 3-5 NEW hook + expansion variations** that:
   - Target different avatars or use different frameworks than the existing copy
   - Use different hook scenes or angles from the manuscript
   - Bring fresh energy — if the existing copy leads with spice, try vulnerability; if it
     leads with the hero, try the heroine's POV
   - Follow all formatting rules from the ad-copy-writer skill

5. **Write 2-3 NEW headlines** following the same purpose categories (genre signal, heroine
   power, hero devotion, scene tension, trope signal) but with fresh angles.

6. **Keep the existing SEO block** unless the author specifically asks to refresh it.

7. **Present the new variations** in the same paste-ready format: each variation is a complete,
   self-contained ad block the author can copy straight into Ads Manager.

8. **Ask if they want to save** the new variations alongside the existing ones or as a
   separate document.
