# Example Ad Copy by Subgenre

These are illustrative examples showing the ad copy formula in action across different romance
subgenres. Use them as reference for tone, structure, and format — adapt the specifics to your
book.

**IMPORTANT: Each variation needs a DIFFERENT hook.** The examples below show one variation
per subgenre, but when writing 5 variations for a single book, each one must open with a
different excerpt, quote, or angle. The hook is the whole point of having variations — if
they all open the same way, there's no reason to test five ads. See the two dark mafia
examples below for contrast.

## What Makes These Examples Work

1. **The hook stops the scroll.** Each one creates a question that demands an answer.
2. **The stakes bridge signals genre.** Cold readers know immediately what kind of book this is.
3. **The trope list is algorithm food AND reader bait.** Each trope is a keyword Facebook uses for targeting.
4. **Social proof uses emotional language.** "I screamed" > "Great book."
5. **The CTA matches the ad destination.** Sales ads push to buy direct. Traffic ads push to retailer choice.
6. **The 5 descriptions each feed the algorithm differently.** Payload, Comps, Vibe Stack, Keyword Sink, and Wildcard — Meta rotates them automatically.
